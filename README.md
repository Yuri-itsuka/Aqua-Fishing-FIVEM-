# Aqua-Fishing-FIVEM

## 🎣 ESX Fishing Complete

A complete fishing system for FiveM ESX servers with upgrade mechanics, NPC interaction, statistics, and more.

---

## 📦 Installation

1. Drag the folder `esx_fishing` into your `resources` folder.
2. Import the `install.sql` file into your database.
3. Add the following to your `server.cfg`:

```cfg
ensure oxmysql
ensure es_extended
ensure esx_fishing
```

---

## ⚠️ Important

If the `Use` button for the fishing rod (`angel`) is greyed out in your inventory,
you must configure the item as `usable/useable` in your inventory system.

---

## ✨ Features

* Fishing rod as usable item
* Fishing only at specific locations
* Fishing spot blips
* Fishing rod animations
* Cancel fishing with `E`
* NPC with interaction menu
* Sell fish system
* Fishing rod upgrade with rare item `angel_update`
* Upgrades increase fish sell rewards
* Top fisherman leaderboard
* Player statistics saved in SQL database
